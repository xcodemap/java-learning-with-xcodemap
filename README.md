## java-learning-with-xcodemap
深入源码理解 Java 技术体系。

* 官网：https://xcodemap.tech/
* B 站：https://space.bilibili.com/3546573562710135

## 为什么需要深入源码
在 AI 时代，程序员的核心能力不再是“写代码”，而是“读代码”。
只有能够全局掌控系统，才能判断优劣、排查问题、消除风险，才能胜任“大模型善后工程师”的角色。

## 学习目标
深入理解系统的行为，洞察其设计意图，而非陷入琐碎的结构细节。

## 环境准备
Idea 插件市场搜索安装 XCodeMap 以便录制程序的执行过程。注意，需要在插件主界面，配置“全录模式”，并且加上依赖包：
* org.spring
* io.netty
* org.apache.ibatis
